# Инструкции для настройки в Unity

## Проблемы, которые нужно исправить:

### 1. Статистика не работает
**Проблема:** Метод `Application.ExternalCall` устарел в новых версиях Unity.

**Решение:** 
- Код уже обновлен для использования `Application.ExternalEval`
- Убедитесь, что в Unity объект с компонентом `CubeManeger` называется **"Cube"** (не "CubeManeger")
- Проверьте, что `GameDataManager` инициализируется правильно

### 2. Автоматическая сборка не начинается
**Проблема:** Метод `SolveCube()` может не вызываться или блокироваться.

**Что проверить:**
1. Убедитесь, что объект с компонентом `CubeManeger` называется **"Cube"** в иерархии Unity
2. Проверьте, что переменные `canShuffle` и `canRotated` установлены в `true`
3. Добавьте логирование в метод `SolveCube()` для отладки

### 3. Настройка материалов для новых стилей
**Что нужно сделать:**
1. Откройте префаб кубика в Unity
2. Найдите компонент `CubePiaceScr`
3. В инспекторе найдите массивы материалов:
   - `Neon Collors` (размер: 7)
   - `Grey Collors` (размер: 7)
4. Назначьте материалы для каждого индекса:
   - Индексы 0-5: материалы для граней (Up, Down, Left, Right, Front, Back)
   - Индекс 6: материал для внутренней части кубика

**Рекомендуемые цвета для Neon:**
- Яркие неоновые цвета: розовый, голубой, зеленый, желтый, оранжевый, красный

**Рекомендуемые цвета для Grey:**
- Оттенки серого: светло-серый, темно-серый, средний серый и т.д.

### 4. Проверка GameDataManager
**Что проверить:**
1. Убедитесь, что `GameDataManager` создается автоматически (использует Singleton pattern)
2. Проверьте, что данные сохраняются в PlayerPrefs для WebGL
3. Добавьте логирование в методы сохранения/загрузки

## Дополнительные улучшения:

1. **Добавьте логирование** в ключевые методы для отладки
2. **Проверьте консоль Unity** на наличие ошибок
3. **Убедитесь**, что все скрипты скомпилированы без ошибок

## Тестирование:

1. Запустите игру в Unity Editor
2. Нажмите T для тестирования автоматической сборки
3. Проверьте консоль на наличие сообщений об ошибках
4. После компиляции для WebGL проверьте работу в браузере
